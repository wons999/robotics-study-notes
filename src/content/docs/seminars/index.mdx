# 세미나

논문 Survey 노트를 발표, 리뷰 미팅, 스터디 공유에 맞게 다시 구성하는 섹션이다. 깊게 읽는 문서는 `논문 Survey`에 두고, 여기서는 발표 흐름, 핵심 그림, 말로 설명할 포인트, 토론 질문을 정리한다.

## 세미나 목록

| 세미나 | 자료 | 상태 |
| --- | --- | --- |
| T-Rex: Tactile-Reactive Dexterous Manipulation | [세미나 페이지](./t-rex/) | 초안 |

## 자료 구성 원칙

| 항목 | 기준 |
| --- | --- |
| 발표 시간 | 35-45분 기준으로 구성 |
| 슬라이드 | 한 장에 하나의 주장만 남긴다 |
| 이미지 | 논문 figure는 발표 흐름에 맞게 재배치한다 |
| 해설 메모 | 슬라이드 본문과 분리할 해석, 근거, 질문을 정리한다 |
| 토론 | 마지막에 연구/시스템 관점 질문을 남긴다 |
