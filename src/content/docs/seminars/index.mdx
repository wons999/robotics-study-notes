# 세미나

논문 Survey 노트를 발표, 리뷰 미팅, 스터디 공유에 맞게 다시 구성하는 섹션이다. 깊게 읽는 문서는 `논문 Survey`에 두고, 여기서는 발표 흐름, 웹 슬라이드, 발표 자료만 관리한다.

## 섹션 역할

| 항목 | 설명 |
| --- | --- |
| 목적 | 논문 하나를 발표 가능한 흐름으로 재구성한다. |
| 포함 자료 | 발표 슬라이드, 발표용 메모, 원문/Survey 링크 |
| 제외 자료 | 논문 전체 세부 분석과 코드 레벨 검토 본문은 `논문 Survey`에 둔다. |
| 사용 방식 | 세미나를 열 때는 이 섹션에서 발표 페이지로 들어가고, 더 깊은 근거 확인은 연결된 Survey로 이동한다. |
