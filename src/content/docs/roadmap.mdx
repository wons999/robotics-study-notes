# 공부 로드맵

## 목표

로봇 시스템에서 perception, control, learning이 어떻게 연결되는지 설명하고, 작은 실험으로 검증하고, 논문 아이디어를 실제 시스템 제약과 비교할 수 있는 사람이 되는 것.

## 1단계: 바로 써먹는 기초

| 주차 | 주제 | 결과물 |
| --- | --- | --- |
| 1 | SE(3), FK, 좌표계 | 2-link/6-DoF pose notebook, frame convention note |
| 2 | Jacobian과 numerical IK | damped least-squares IK 구현 |
| 3 | Dynamics와 control | joint PD, computed torque, impedance control 정리 |
| 4 | Contact와 manipulation 기초 | grasp/contact 용어, wrench/friction cone 요약 |

## 2단계: Manipulation과 로봇 시스템

| 주제 | 질문 |
| --- | --- |
| Manipulation | 어떤 제약 때문에 조작 과제가 어려워지는가? |
| Sensing | closed-loop behavior에 실제로 도움이 되는 관측은 무엇인가? |
| Actuation | bandwidth, compliance, limit가 controller를 어떻게 제한하는가? |
| System evaluation | 데모 성공률 너머의 진짜 진전을 어떤 metric으로 볼 수 있는가? |

## 3단계: Robot Learning Follow-up

| 주제 | 따라갈 내용 |
| --- | --- |
| Imitation learning | data collection, teleoperation, action representation |
| RL | reward design, sim-to-real, residual policy, contact-rich task |
| Diffusion policy | action chunking, temporal consistency, latency |
| VLA | vision-language grounding, low-level action decoding, cross-embodiment transfer |

## 주간 루프

1. 개념이나 논문 하나를 고른다.
2. 한 페이지짜리 직관 노트를 쓴다.
3. 아주 작은 구현이나 재현 실험을 만든다.
4. 로봇 시스템 관점의 실전 의미를 뽑는다.
5. 다음 주에 이어갈 질문을 남긴다.

## 읽을 자료 큐

- Modern Robotics: mechanics, planning, control.
- Underactuated Robotics: dynamics, planning, learning for control.
- Robot manipulation learning survey.
- VLA survey와 최신 manipulation benchmark.
