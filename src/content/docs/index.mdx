## 현재 초점

이 사이트는 기초 로보틱스와 최신 robot learning 연구를 같은 공간에서 연결하기 위한 작업 노트입니다. 목표는 논문을 모으는 것이 아니라, 시스템을 이해하고 작은 실험으로 검증할 수 있는 판단력을 쌓는 것입니다.

<CardGrid>
  <Card title="로보틱스 기초" icon="seti:config">
    FK, IK, Jacobian, dynamics, trajectory generation, impedance control, whole-body control을 차근차근 정리합니다.
  </Card>
  <Card title="로봇 시스템" icon="seti:folder">
    sensing, actuation, manipulation, control architecture, evaluation을 시스템 관점에서 정리합니다.
  </Card>
  <Card title="로봇 러닝" icon="seti:python">
    imitation learning, RL, diffusion policy, VLA, multimodal policy learning을 구현 관점으로 따라갑니다.
  </Card>
  <Card title="논문 노트" icon="seti:markdown">
    각 논문을 문제 설정, 데이터셋, 방법론, 모델 구조, 실험 결과, 후속 논문 흐름으로 정리합니다.
  </Card>
</CardGrid>

## 운영 원칙

매주 하나의 주제를 잡고 `개념 -> 수식 -> 작은 구현 -> 논문 연결 -> 후속 질문` 순서로 정리합니다. 좋은 노트는 나중에 다시 읽었을 때 “무엇을 이해했고, 어떤 실험으로 확인할 수 있지?”라는 질문에 답해야 합니다.
