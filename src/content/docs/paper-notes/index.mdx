# 논문 Survey

이 섹션은 로보틱스, robot learning, VLA, tactile policy, flow/diffusion action model 관련 논문을 깊게 읽고 재사용 가능한 형태로 정리하는 공간이다. 단순 요약보다 문제 설정, 데이터셋, 방법론, 모델 구조, 학습 recipe, 실험 결과, appendix, 코드 구현 차이를 함께 남기는 것을 기준으로 한다.

## 섹션 역할

| 항목 | 설명 |
| --- | --- |
| 목적 | 논문을 다시 열지 않아도 핵심 방법론과 수치, 구현 포인트를 확인할 수 있게 정리한다. |
| 포함 자료 | 논문별 Survey, model family overview, citation 흐름, appendix notes, 공식 코드 확인 메모 |
| 작성 기준 | `rules/survey.md`의 high-effort Survey 규칙을 따른다. |
| 세미나와의 관계 | 발표 흐름은 `세미나` 섹션에 두고, 근거와 상세 분석은 이 섹션에 둔다. |
