# EgoScale

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | EgoScale: Scaling Dexterous Manipulation with Diverse Egocentric Human Data |
| 저자 | Ruijie Zheng, Dantong Niu, Yuqi Xie, Jing Wang, Mengda Xu, Yunfan Jiang, Fernando Castañeda, Fengyuan Hu, You Liang Tan, Letian Fu, Trevor Darrell, Furong Huang, Yuke Zhu, Danfei Xu, Linxi Fan |

## 한 줄 요약

EgoScale은 20,854시간 규모의 action-labeled egocentric human video로 dexterous manipulation prior를 학습하고, lightweight human-robot mid-training으로 robot hand에 transfer하는 framework다.

## 이 논문의 위치

EgoScale은 human egocentric data에서 손-물체 상호작용 prior를 얻어 dexterous manipulation으로 옮기려는 data scaling 논문이다. robot data가 비싸고 적을 때, human video prior를 어떻게 활용할지 보는 기준점이 된다.

| 관점 | EgoScale이 강조하는 점 | 읽을 때 확인할 질문 |
| --- | --- | --- |
| pretraining | 대규모 human egocentric video | human hand prior가 robot hand에 얼마나 transfer되는가 |
| robot alignment | aligned human-robot mid-training | embodiment gap을 어떤 데이터로 줄이는가 |
| sensory gap | vision 중심 human prior | force, tactile, proprioception 부재를 어떻게 보완하는가 |
| task | dexterous manipulation | human data scale과 downstream success가 어떻게 연결되는가 |

## 데이터셋

EgoScale의 핵심은 scale이다. 논문은 20,854시간 이상의 action-labeled egocentric human video를 사용한다고 설명한다. 이 데이터는 robot data보다 훨씬 모으기 쉽고, 손-물체 상호작용의 다양성이 크다.

중요한 점은 이 데이터가 robot action data가 아니라는 것이다. 사람 손의 움직임과 로봇 손의 kinematics/action space는 다르다. 따라서 EgoScale은 pretraining 후 robot alignment 단계가 필요하다.

## 모델 구조

| 구성 | 역할 |
| --- | --- |
| VLA backbone | egocentric video와 language/action label에서 visuomotor prior 학습 |
| human pretraining stage | 사람 손-물체 상호작용의 large-scale pattern 학습 |
| human-robot mid-training | human prior를 robot embodiment에 맞게 정렬 |
| downstream adaptation | 적은 robot supervision으로 target task 적응 |

## 방법론

EgoScale은 “robot data가 부족하면 human data를 어떻게 쓸 것인가?”라는 질문에 답한다. 핵심은 두 단계다.

1. human egocentric data로 넓은 motor prior를 배운다.
2. robot data로 embodiment mismatch를 줄인다.

논문은 human data scale과 validation loss 사이의 scaling law, 그리고 validation loss와 downstream robot performance의 상관을 강조한다. 즉, human data scale이 단순히 커지는 것 이상의 예측 가능한 효과가 있다고 주장한다.

## 기억할 점

- EgoScale은 human egocentric data scaling의 대표 사례다.
- 하지만 tactile이 없는 human video만으로는 force, slip, deformation을 배울 수 없다.
- human prior는 유용하지만 robot action, force, tactile, proprioception으로 바로 이어지지는 않는다.

## 한계

- egocentric video에는 robot proprioception, tactile, force가 없다.
- 사람 손과 로봇 손의 morphology 차이가 크다.
- contact-rich precision task에는 robot tactile data가 추가로 필요하다.
