# OmniVTLA

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | OmniVTLA: Vision-Tactile-Language-Action Model with Semantic-Aligned Tactile Sensing |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2508.08706 |
| 태그 | tactile, VLA, semantic alignment, tactile encoder |

## 한 줄 요약

OmniVTLA는 tactile signal을 VLA의 semantic space와 맞추기 위해 dual tactile encoder와 ObjTac dataset을 사용하는 tactile-aware VLA다.

## 데이터셋과 문제 설정

ObjTac은 tactile, vision, text를 함께 다루는 tri-modal dataset이다. 목적은 tactile signal이 단순 sensor value가 아니라 object/property/action semantics와 연결되게 만드는 것이다.

## 방법론과 모델 구조

| 구성 | 역할 |
| --- | --- |
| pretrained ViT tactile path | tactile observation을 vision feature와 비슷한 공간으로 처리 |
| semantic-aligned tactile ViT | tactile-text/vision alignment를 강화 |
| VLA policy | aligned tactile feature를 manipulation action에 사용 |

## T-Rex와의 관계

OmniVTLA는 tactile semantic alignment에 가깝고, T-Rex는 tactile reactivity와 control frequency에 가깝다. 둘을 함께 보면 tactile이 “무엇을 의미하는가”와 “얼마나 빨리 반응할 것인가”를 분리해서 볼 수 있다.
