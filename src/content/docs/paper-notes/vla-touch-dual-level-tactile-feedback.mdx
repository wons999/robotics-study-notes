# VLA-Touch

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | VLA-Touch: Enhancing Vision-Language-Action Models with Dual-Level Tactile Feedback |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2507.17294 |
| 태그 | tactile, VLA, diffusion controller, tactile-language model |

## 한 줄 요약

VLA-Touch는 base VLA를 크게 바꾸지 않고, tactile-language planning feedback과 tactile-conditioned diffusion controller를 붙여 contact-rich manipulation을 보강한다.

## 방법론

| 수준 | 역할 |
| --- | --- |
| high-level tactile feedback | tactile-language model이 task planning에 필요한 접촉 의미를 제공 |
| low-level tactile feedback | diffusion controller가 VLA action을 tactile signal로 보정 |

## T-Rex와의 관계

VLA-Touch는 modular approach다. 기존 VLA 주변에 tactile module을 붙인다. T-Rex는 integrated approach다. MoT 안에 tactile expert를 넣고 tactile-rich dataset으로 함께 학습한다.

## 기억할 점

- tactile은 semantic planning과 execution refinement 두 수준 모두에서 유용하다.
- frozen VLA를 살리는 전략과 end-to-end tactile-reactive policy 전략을 비교할 때 좋은 reference다.
