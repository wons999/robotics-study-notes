# T-Rex 읽기 흐름

이 페이지는 **T-Rex: Tactile-Reactive Dexterous Manipulation**을 중심으로 논문 그래프를 따라가기 위한 지도다. 목표는 논문 하나를 고립해서 요약하는 것이 아니라, 어떤 논문이 어떤 문제를 열었고 다음 논문이 그 문제를 어떻게 이어받는지 추적하는 것이다.

## 읽기 규칙

| 깊이 | 의미 | 정리 방식 |
| --- | --- | --- |
| 0 | 시작 논문 | 자세한 한국어 노트 작성 |
| 1 | 시작 논문의 직접 비교군 또는 가까운 선행 연구 | 논문별 survey card 작성 |
| 2 | 방법론, 데이터셋, model family의 기반 논문 | compact note 작성 후 필요하면 확장 |
| 이후 | citation을 따라 계속 확장 | 실제 이해에 필요해질 때 추가 |

## 다음 확장 후보

- ACT와 Diffusion Policy: action chunking과 imitation learning baseline 이해에 필요하다.
- EgoMimic, DexWild, EgoDex, EgoVLA: human video에서 robot action으로 넘어가는 흐름을 이해하는 데 필요하다.
- Octo, DROID, BridgeData v2: robot data scaling과 cross-embodiment learning의 데이터 생태계를 이해하는 데 필요하다.
- Pinocchio, Pink, CasADi: 논문 survey보다는 실제 로봇 제어 stack을 이해할 때 필요하다.
