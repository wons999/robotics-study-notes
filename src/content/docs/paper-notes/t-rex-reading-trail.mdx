# T-Rex Reading Trail

This page tracks the paper graph around **T-Rex: Tactile-Reactive Dexterous Manipulation**. The goal is to avoid isolated paper summaries: every note should record which earlier papers it depends on and which papers should be followed next.

## Trail Policy

| Depth | Meaning | Action |
| --- | --- | --- |
| 0 | Seed paper | Write a detailed note. |
| 1 | Direct conceptual or baseline papers used by the seed | Write one note per paper. |
| 2 | Foundational methods, datasets, or model families behind depth-1 papers | Write compact notes first, expand when studied. |
| Later | Recursive expansion | Add when a paper becomes important enough to affect understanding. |
