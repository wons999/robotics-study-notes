## One-line Summary

GR00T N1 is a dual-system humanoid foundation model that couples a vision-language reasoning module with a fast diffusion-transformer action module.

## Problem Setting

- Robot embodiment: humanoid robots and multiple robot embodiments in simulation and real tasks.
- Observation: vision and language instructions.
- Action: continuous motor actions generated by a fast action module.
- Data: real robot trajectories, human videos, and synthetic data.
- Task: language-conditioned bimanual manipulation and generalist robot behavior.

## Core Idea

GR00T N1 separates slow perception/reasoning from fast motor action. This makes it relevant to T-Rex because both papers are concerned with the latency mismatch between high-level multimodal reasoning and low-level control.

## Method Details

- Architecture: System 2 vision-language module plus System 1 diffusion transformer action module.
- Training recipe: heterogeneous data mixture and end-to-end coupling.
- Data: real robot, human video, and synthetic sources.
- Inference / control loop: slow reasoning conditions fast motor generation.

## What I Should Remember

- GR00T N1 is a humanoid-oriented foundation model.
- T-Rex uses a different split: low-frequency visual action generation plus high-frequency tactile refinement.
- Both papers support the broader trend toward variable-rate or dual-system robot foundation models.

## Limitations

- Tactile feedback is not the main focus.
- Humanoid generality does not automatically solve contact-rich dexterity.
- Dataset mixture and embodiment details heavily shape transfer.
