# GR00T N1

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | GR00T N1: An Open Foundation Model for Generalist Humanoid Robots |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2503.14734 |
| 태그 | humanoid, VLA, diffusion transformer, foundation model |

## 한 줄 요약

GR00T N1은 vision-language reasoning module과 빠른 action module을 결합한 humanoid robot foundation model이다.

## 방법론

slow reasoning과 fast motor generation을 분리한다. 이 구조는 무거운 multimodal reasoning과 빠른 low-level control 사이의 속도 차이를 다루려는 시도다.

## 이 논문의 위치

GR00T N1은 humanoid generalist policy 관점에서 foundation model을 어떻게 구성할지 보여주는 논문이다. 특히 큰 reasoning model과 빠른 execution pathway를 분리하는 fast-slow 구조를 통해, 로봇 policy가 하나의 속도로 모든 것을 처리하기 어렵다는 현실을 드러낸다.
