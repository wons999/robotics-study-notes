# VLA Survey Map

## 여기서 말하는 VLA

VLA는 visual observation과 language instruction을 받아 robot action을 생성하는 모델 계열이다. Manipulation에서 어려운 점은 semantic understanding만이 아니다. 실제 로봇에서는 embodiment constraint, latency, contact, safety, controller interface가 함께 들어온다.

## 읽을 때 기록할 축

| 축 | 기록할 내용 |
| --- | --- |
| Architecture | monolithic, hierarchical, planner-policy split, action decoder |
| Data | Open X-Embodiment, teleoperation, human video, synthetic data |
| Action representation | discrete token, continuous delta, action chunk, flow/diffusion |
| Embodiment | single arm, dual arm, mobile base, humanoid, dexterous hand |
| Evaluation | real robot success, generalization, recovery, safety |

## 읽기 질문

- 모델이 finger까지 직접 제어하는가, 아니면 arm/gripper command만 내는가?
- policy가 closed-loop manipulation에 충분히 빠른가?
- camera, calibration, robot morphology에 어떤 가정을 두는가?
- 성능이 data scale에서 온 것인가, controller quality에서 온 것인가?
- 작은 simulator나 toy setup에서 먼저 검증할 수 있는 아이디어인가?
