# A Touch, Vision, and Language Dataset

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | A Touch, Vision, and Language Dataset for Multimodal Alignment |
| 연도 | ICML, 2024 |
| 링크 | https://arxiv.org/abs/2402.13232 |
| 태그 | tactile, vision-language, multimodal alignment, dataset |

## 한 줄 요약

이 논문은 tactile reading을 image와 language에 맞춰 alignment할 수 있는 dataset과 tactile encoder 학습 방향을 제시한다.

## 데이터셋

touch와 vision pair를 모으고, human label과 VLM 기반 pseudo-label을 사용해 language와 연결한다. 목적은 tactile signal이 “거칠다/미끄럽다/딱딱하다” 같은 의미와 연결되는 representation을 갖게 하는 것이다.

## 이 논문의 위치

Touch-vision-language dataset 논문들은 tactile signal을 semantic reasoning과 어떻게 맞출 수 있는지 보여준다. tactile이 단순 control feedback을 넘어 “무엇을 만지고 있는가”, “재질/상태가 무엇인가”를 설명하는 modality가 될 수 있는지 확인하는 축이다.
