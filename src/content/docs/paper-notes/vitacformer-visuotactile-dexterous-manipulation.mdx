# ViTacFormer

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | ViTacFormer: Learning Cross-Modal Representation for Visuo-Tactile Dexterous Manipulation |
| 저자 | Liang Heng, Haoran Geng, Kaifeng Zhang, Pieter Abbeel, Jitendra Malik |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2506.15953 |
| 태그 | tactile, dexterous manipulation, cross-modal representation, future tactile prediction |

## 한 줄 요약

ViTacFormer는 vision과 touch를 cross-attention으로 결합하고 future tactile prediction을 auxiliary objective로 사용해, multi-fingered dexterous manipulation에 필요한 visuo-tactile representation을 학습한다.

## 이 논문의 위치

ViTacFormer는 tactile을 단순한 보조 센서가 아니라 dexterous manipulation representation의 핵심 modality로 다룬다. tactile-reactive control보다 “vision과 touch를 어떤 latent로 융합할 것인가”에 초점이 있는 논문이다.

| 관점 | ViTacFormer가 강조하는 점 | 읽을 때 확인할 질문 |
| --- | --- | --- |
| 핵심 질문 | vision과 tactile을 어떤 latent로 융합할까? | fused latent가 실제 action 선택에 얼마나 영향을 주는가 |
| 주요 구조 | cross-attention encoder + tactile prediction head | 미래 tactile 예측이 contact dynamics를 잡는가 |
| 데이터 관점 | dexterous task demonstration | curriculum이 representation 안정성에 어떤 역할을 하는가 |

## 문제 설정

| 항목 | 내용 |
| --- | --- |
| 로봇 | anthropomorphic multi-fingered hand |
| observation | high-resolution vision + tactile signal |
| action | dexterous hand manipulation action |
| task | precision manipulation, long-horizon multi-stage manipulation |
| 어려움 | 손가락 접촉은 occlusion이 많고, vision alone으로는 접촉 상태를 알기 어렵다 |

## 데이터와 curriculum

ViTacFormer는 easy-to-challenging curriculum을 사용한다. 쉬운 task에서 시작해 점점 정밀하고 긴 조작으로 representation을 다듬는다.

이 방식의 의미는 중요하다. tactile representation은 단순히 큰 모델에 데이터를 많이 넣는다고 생기지 않는다. 접촉 난이도와 과제 구조를 점진적으로 올리면, 모델이 visual feature와 tactile feature를 더 안정적으로 연결할 수 있다.

## 모델 구조

| 구성 | 역할 |
| --- | --- |
| visual encoder | object, hand pose, scene context를 추출 |
| tactile encoder | fingertip contact signal을 latent로 변환 |
| cross-attention encoder | vision과 touch가 서로 참고하며 fused representation 생성 |
| autoregressive tactile prediction head | 미래 tactile signal을 예측해 contact dynamics를 latent에 압축 |
| imitation policy | fused representation으로 dexterous action 생성 |

## 방법론

핵심은 future tactile prediction이다. 지금의 vision/tactile observation만 잘 맞추는 것이 아니라, 앞으로 어떤 tactile signal이 생길지 예측하게 하면 모델이 contact transition을 더 잘 이해하게 된다.

예를 들어 물체를 잡는 순간, vision은 물체 위치를 보여주지만 “곧 미끄러질지”, “손가락 압력이 충분한지”는 tactile future가 더 잘 알려준다. ViTacFormer는 이런 contact dynamics를 representation learning objective에 넣는다.

## 실험에서 봐야 할 점

- vision-only와 tactile-only 대비 fused representation이 얼마나 좋은가?
- future tactile prediction head를 제거하면 성능이 얼마나 떨어지는가?
- long-horizon dexterous task에서 몇 단계까지 안정적으로 유지되는가?
- precision task에서 tactile이 실패 유형을 어떻게 줄이는가?

## 기억할 점

- tactile은 raw sensor value를 그대로 넣기보다 representation learning objective와 함께 설계해야 한다.
- future tactile prediction은 contact transition을 latent에 담게 만드는 강한 auxiliary task다.
- vision-touch fusion은 occlusion이 많은 dexterous manipulation에서 특히 중요하다.

## 한계

- VLA scale의 language-conditioned generalization보다는 dexterous representation learning에 가깝다.
- tactile representation은 hand/sensor configuration에 민감할 수 있다.
- high-frequency asynchronous control 구조 자체가 중심은 아니다.
