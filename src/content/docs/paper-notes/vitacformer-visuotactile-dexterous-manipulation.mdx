# ViTacFormer

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | ViTacFormer: Learning Cross-Modal Representation for Visuo-Tactile Dexterous Manipulation |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2506.15953 |
| 태그 | tactile, dexterous manipulation, cross-modal representation |

## 한 줄 요약

ViTacFormer는 vision과 touch가 서로 attend하는 representation을 학습하고, future tactile prediction을 통해 dexterous manipulation에 필요한 접촉 정보를 더 잘 담도록 만든다.

## 데이터셋과 문제 설정

대상은 multi-fingered dexterous hand가 수행하는 정밀 조작이다. observation은 vision과 tactile이고, demonstration을 통해 imitation policy를 학습한다.

## 방법론과 모델 구조

핵심은 tactile을 raw state로만 쓰지 않고, vision과 결합된 latent representation으로 만드는 것이다.

| 구성 | 역할 |
| --- | --- |
| visual encoder | object와 scene context 추출 |
| tactile encoder | 손가락 접촉 상태 추출 |
| cross-attention | vision-touch 상호작용을 latent feature로 통합 |
| future tactile prediction | 앞으로의 contact 변화를 예측하게 하여 representation을 강화 |

## T-Rex와의 관계

ViTacFormer는 tactile representation learning 쪽에 가깝고, T-Rex는 여기에 VLA pretraining, tactile mid-training, high-frequency tactile expert를 얹은 더 큰 시스템이다.

## 기억할 점

- tactile을 policy input으로 넣는 것보다 representation을 어떻게 만들지가 중요하다.
- future tactile prediction은 contact dynamics를 latent space에 넣는 좋은 auxiliary objective다.
