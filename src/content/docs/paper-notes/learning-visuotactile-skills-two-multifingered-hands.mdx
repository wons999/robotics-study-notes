# Learning Visuotactile Skills with Two Multifingered Hands

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Learning Visuotactile Skills with Two Multifingered Hands |
| 연도 | arXiv, 2024 |
| 링크 | https://arxiv.org/abs/2404.16823 |
| 태그 | bimanual manipulation, multifingered hand, tactile, teleoperation |

## 한 줄 요약

이 논문은 두 개의 multifingered hand로 visuotactile demonstration을 수집하고, 이를 이용해 bimanual dexterous skill을 학습하는 시스템을 다룬다.

## 데이터셋과 시스템

핵심은 low-cost teleoperation system인 HATO와 visuotactile demonstration collection이다. dexterous hand 연구에서는 모델보다 데이터 수집 시스템이 병목인 경우가 많기 때문에 중요하다.

## T-Rex와의 관계

T-Rex도 bimanual dexterous tactile data collection에 크게 의존한다. 이 논문은 T-Rex dataset을 보기 전에 “어떻게 양손 multifinger tactile demo를 모을 수 있는가”를 이해하는 데 도움이 된다.
