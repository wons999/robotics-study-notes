# pi_0: Vision-Language-Action Flow Model

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | pi_0: A Vision-Language-Action Flow Model for General Robot Control |
| 연도 | arXiv, 2024 |
| 링크 | https://arxiv.org/abs/2410.24164 |
| 태그 | VLA, flow matching, continuous action, robot foundation model |

## 한 줄 요약

pi_0는 pretrained vision-language backbone과 flow matching action expert를 결합해 continuous robot action을 생성하는 generalist robot policy다.

## 방법론과 모델 구조

| 구성 | 역할 |
| --- | --- |
| VLM backbone | image와 language instruction 처리 |
| flow action expert | continuous action trajectory 생성 |
| robot data mixture | 여러 robot embodiment와 task에서 action distribution 학습 |

## T-Rex와의 관계

T-Rex는 pi_0 계열의 flow matching action generation을 tactile-reactive하게 확장한다. pi_0가 coarse continuous action generation의 기반이라면, T-Rex는 그 action을 tactile expert가 빠르게 보정하는 구조다.

## 기억할 점

- VLA action을 token으로만 다루지 않고 continuous distribution으로 생성할 수 있다.
- tactile feedback이 없으면 contact-rich dexterity에는 여전히 약할 수 있다.
