# VQ-VAE

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Neural Discrete Representation Learning |
| 저자 | Aaron van den Oord, Oriol Vinyals, Koray Kavukcuoglu |
| 연도 | NeurIPS, 2017 |
| 링크 | https://arxiv.org/abs/1711.00937 |
| 태그 | VQ-VAE, discrete latent, vector quantization, representation learning |

## 한 줄 요약

VQ-VAE는 encoder output을 continuous Gaussian latent가 아니라 codebook의 discrete vector로 quantize하여, compact한 token-like latent representation을 학습하는 generative model이다.

## 이 논문의 위치

VQ-VAE는 image, audio, video, tactile/force history처럼 연속적이고 noisy한 신호를 discrete token-like latent로 바꾸는 대표 방법이다. transformer나 autoregressive model이 sensor sequence를 다루도록 만들 때 자주 등장하는 기반 논문이다.

## 직관

연속적인 force signal은 빠르고 noisy하다. 예를 들어 grasp 중 force가 조금씩 변하는 모든 값을 raw로 넣으면 모델이 noise와 의미 있는 contact pattern을 구분하기 어렵다. VQ-VAE는 이런 signal을 codebook token으로 묶어준다.

| raw force history | VQ-VAE tactile token |
| --- | --- |
| 연속값, noisy, sensor drift 영향 | discrete pattern, compact, sequence modeling에 적합 |
| 미세 변화까지 보존 가능 | 정보 손실 가능 |
| 모델 입력이 커질 수 있음 | token 수를 줄일 수 있음 |

## 모델 구조

| 구성 | 역할 |
| --- | --- |
| encoder | input을 latent vector로 변환 |
| codebook | latent vector를 가장 가까운 discrete code로 quantize |
| decoder | discrete code에서 input을 복원 |
| commitment/codebook loss | encoder와 codebook이 안정적으로 맞춰지도록 학습 |

## 방법론

VQ-VAE는 latent를 discrete code로 강제한다. 이 덕분에 autoregressive prior나 transformer가 token sequence처럼 다루기 쉬워진다. 원래 논문은 image, video, speech 같은 modality에서 discrete representation의 장점을 보여준다.

## sensor sequence에서의 역할

로봇 tactile/force sequence에 VQ-VAE를 적용하면 빠르게 변하는 raw signal을 compact한 contact pattern token으로 바꿀 수 있다.

| 적용 대상 | 기대 효과 | 주의할 점 |
| --- | --- | --- |
| force history | noisy continuous signal을 discrete contact pattern으로 압축 | 미세한 force 차이를 잃을 수 있음 |
| tactile deformation sequence | 접촉 형상 변화를 token sequence로 표현 | codebook이 sensor/hand configuration에 과적합될 수 있음 |
| video/action latent | transformer가 다루기 쉬운 token stream 제공 | codebook collapse를 확인해야 함 |

중요한 질문은 tokenization이 정보를 얼마나 줄이면서도 downstream policy에는 충분한 contact state를 남기는가다.

## 읽을 때 주의할 점

- quantization은 정보를 줄이는 대신 안정성을 준다.
- codebook이 잘 학습되지 않으면 특정 code만 쓰는 collapse가 생길 수 있다.
- tactile force의 미세한 continuous variation이 중요한 task에서는 token granularity가 중요하다.
