# VQ-VAE

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Neural Discrete Representation Learning |
| 연도 | NeurIPS, 2017 |
| 링크 | https://arxiv.org/abs/1711.00937 |
| 태그 | VQ-VAE, discrete latent, representation learning |

## 한 줄 요약

VQ-VAE는 encoder output을 continuous latent가 아니라 codebook의 discrete vector로 quantize하여 compact한 token-like representation을 학습한다.

## T-Rex와의 관계

T-Rex는 fingertip force history를 VQ-VAE 방식으로 discrete tactile token으로 압축한다. 빠르고 noisy한 force stream을 그대로 쓰기보다 token화하면 temporal contact pattern을 더 안정적으로 표현할 수 있다.

## 기억할 점

- discrete token은 compact하고 sequence model에 넣기 쉽다.
- 너무 거칠게 quantize하면 미세한 force 변화가 사라질 수 있다.
- codebook collapse 같은 학습 안정성 문제를 주의해야 한다.
