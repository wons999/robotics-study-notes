# Tactile-VLA

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Tactile-VLA: Unlocking Vision-Language-Action Model's Physical Knowledge for Tactile Generalization |
| 저자 | Jialei Huang, Shuo Wang, Fanqi Lin, Yihang Hu, Chuan Wen, Yang Gao |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2507.09160 |
| 태그 | tactile, VLA, physical reasoning, hybrid position-force control |

## 한 줄 요약

Tactile-VLA는 pretrained VLA가 이미 가진 물리적 상식을 tactile sensing과 연결해, 접촉이 중요한 task에서 instruction following과 adaptive reasoning을 가능하게 하려는 framework다.

## 이 논문의 위치

Tactile-VLA는 VLA가 물리적 접촉과 재질, 힘의 변화를 충분히 이해하지 못한다는 문제에서 출발한다. tactile feedback을 language/vision reasoning과 연결해 physical knowledge를 보강하려는 논문이다.

| 관점 | Tactile-VLA가 강조하는 점 | 읽을 때 확인할 질문 |
| --- | --- | --- |
| tactile의 역할 | VLA reasoning과 physical commonsense를 activate | tactile이 semantic reasoning과 control 중 어디에 쓰이는가 |
| controller | hybrid position-force controller 강조 | model intent가 실제 force behavior로 잘 변환되는가 |
| 데이터 | 적은 demonstration으로 tactile grounding | tactile grounding이 task distribution 밖에서도 유지되는가 |

## 문제 설정

| 항목 | 내용 |
| --- | --- |
| task | tactile-aware instruction following, contact-rich manipulation |
| observation | vision, language, tactile feedback |
| action | VLA intent를 hybrid position-force controller로 실행 |
| 핵심 난점 | VLA의 semantic knowledge와 실제 force/tactile interaction 사이의 gap |

## 방법론

Tactile-VLA는 pretrained VLA가 가진 “물리 상식”을 tactile feedback과 연결하려 한다. 예를 들어 “부드럽게 누르기”, “단단히 잡기”, “미끄러지면 다시 조정하기” 같은 개념은 language/vision pretraining 안에 어느 정도 들어 있을 수 있지만, 로봇이 실제 tactile sensor를 통해 그 상태를 인식하고 action에 반영해야만 쓸 수 있다.

## 모델 구조

| 구성 | 역할 |
| --- | --- |
| VLA backbone | image와 instruction을 기반으로 task intent 생성 |
| tactile fusion module | tactile state를 VLA의 reasoning/action generation에 연결 |
| tactile reasoning module | 접촉 feedback에 따라 strategy를 조정 |
| hybrid position-force controller | model intent를 contact-sensitive action으로 변환 |

## 실험에서 봐야 할 점

- tactile feedback이 없을 때와 있을 때 instruction following이 얼마나 달라지는가?
- tactile-relevant commonsense가 zero-shot으로 드러나는가?
- 적은 demonstration만으로 tactile grounding이 가능한가?
- controller가 policy의 모호한 intent를 실제 force behavior로 잘 바꾸는가?

## 기억할 점

- Tactile-VLA는 tactile을 “VLA의 physical knowledge를 깨우는 grounding signal”로 본다.
- 단순 tactile concatenation과 구조적 tactile module의 차이를 봐야 한다.
- tactile-aware VLA는 reasoning, planning, control 중 어느 레벨을 고치는지 명확히 구분해야 한다.

## 한계

- high-frequency tactile reactivity 자체가 핵심인 논문은 아니다.
- tactile grounding이 sensor와 task distribution에 의존할 수 있다.
- hybrid controller가 잘 설계되어야 VLA의 intent가 실제 접촉 행동으로 바뀐다.
