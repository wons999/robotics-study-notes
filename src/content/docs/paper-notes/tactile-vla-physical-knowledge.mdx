# Tactile-VLA

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Tactile-VLA: Unlocking Vision-Language-Action Model's Physical Knowledge for Tactile Generalization |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2507.09160 |
| 태그 | tactile, VLA, physical reasoning, force control |

## 한 줄 요약

Tactile-VLA는 pretrained VLA가 가진 물리적 상식을 tactile feedback과 연결해 contact-rich task에서 더 잘 일반화하게 만들려는 시도다.

## 데이터셋과 문제 설정

문제는 language-conditioned robot manipulation에서 접촉 상태가 중요한 경우다. vision-language만으로는 힘, 압력, 미끄러짐을 충분히 알 수 없기 때문에 tactile signal을 모델과 controller에 연결한다.

## 방법론과 모델 구조

| 구성 | 역할 |
| --- | --- |
| VLA backbone | instruction과 visual context 처리 |
| tactile fusion | 접촉 정보를 정책 입력에 통합 |
| hybrid position-force controller | VLA output을 접촉 상황에 맞는 실행으로 변환 |

## T-Rex와의 관계

Tactile-VLA는 tactile을 VLA reasoning에 연결하는 쪽에 강하고, T-Rex는 tactile을 high-frequency correction signal로 쓰는 쪽에 더 강하다. 둘은 tactile-aware VLA라는 같은 문제를 다른 속도/구조 관점에서 푼다.

## 기억할 점

- VLA에 tactile을 넣는 목적은 semantic reasoning뿐 아니라 물리적 interaction grounding이다.
- 다만 tactile을 넣는 것만으로 충분하지 않고, timing과 architecture가 중요하다.
