# Reactive Diffusion Policy

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Reactive Diffusion Policy: Slow-Fast Visual-Tactile Policy Learning for Contact-Rich Manipulation |
| 연도 | RSS, 2025 |
| 링크 | https://arxiv.org/abs/2503.02881 |
| 태그 | tactile, diffusion policy, slow-fast control, imitation learning |

## 한 줄 요약

RDP는 느린 latent diffusion policy와 빠른 tactile-conditioned controller를 결합해, action chunk 실행 중에도 접촉 feedback에 반응하도록 만든 정책이다.

## 데이터셋과 문제 설정

데이터는 tactile feedback을 포함한 teleoperation demonstration에서 온다. 핵심 과제는 RGB만 보고는 접촉 상태를 충분히 알기 어려운 contact-rich manipulation이다.

| 항목 | 내용 |
| --- | --- |
| observation | vision + tactile/force feedback |
| action | latent diffusion으로 만든 action chunk와 tactile correction |
| task | 삽입, 접촉 유지, 미끄러짐 보정처럼 빠른 feedback이 필요한 조작 |

## 방법론과 모델 구조

RDP는 policy를 두 속도로 나눈다. slow policy는 diffusion으로 action chunk를 만들고, fast tactile pathway는 실행 중 들어오는 touch를 이용해 행동을 조정한다.

T-Rex와의 관계는 매우 직접적이다. RDP가 “slow visual plan + fast tactile correction”이라는 구조적 문제를 먼저 보여줬고, T-Rex는 이 아이디어를 VLA/MoT와 대규모 tactile mid-training으로 확장한다.

## 기억할 점

- tactile feedback은 action chunk 시작 시점에만 넣으면 늦다.
- diffusion policy의 장점인 multimodal action modeling과 tactile reactivity를 함께 쓰려는 시도다.
- T-Rex를 이해할 때 RDP는 가장 가까운 구조적 선행 연구 중 하나다.

## 한계

- generalist VLA라기보다는 task/data-specific imitation policy에 가깝다.
- T-Rex처럼 human egocentric pretraining이나 대규모 tactile robot mid-training을 쓰지는 않는다.
