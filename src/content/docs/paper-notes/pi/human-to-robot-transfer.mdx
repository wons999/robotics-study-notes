## 한 줄 요약

이 연구는 egocentric human video를 3D hand-position action이 있는 또 하나의 embodiment처럼 co-finetune하면, 별도 transfer mechanism 없이도 π0.5 scale에서 human-to-robot transfer가 나타나고, human-only scenario generalization에서 약 2배 성능 향상을 얻는다는 결과다.

## 문제의식

Robot trajectory는 비싸고 느리며, task/site마다 새로 모으기 어렵다. 반대로 human egocentric video는 쉽게 모을 수 있지만 robot과 morphology/action space가 다르다. 기존 연구는 image masking, human hand를 robot hand로 바꾸는 generative conversion, humanoid hardware alignment 같은 별도 transfer mechanism을 쓰기도 한다. PI 연구의 질문은 더 단순하다.

> robot foundation model을 충분히 크게 pretrain하면, human video를 특별한 alignment 없이도 fine-tuning data source로 흡수할 수 있는가?

## Method: Simple Human-Robot Co-Finetuning

| 구성 | 공식 글 기준 내용 |
| --- | --- |
| base model | full π0.5 |
| human data | egocentric wearable-camera human videos |
| human action label | 3D hand positions |
| robot data | task와 관련된 robot demonstrations |
| recipe | human video를 기존 robot embodiment 중 하나처럼 취급해 robot data와 mixture fine-tuning |
| special mechanism | masking/conversion/humanoid matching 없이 진행했다고 설명 |

이 접근의 포인트는 “human video에서 robot action을 직접 복원했다”가 아니다. Human video는 object ordering, affordance, task-specific arrangement, scene strategy를 제공하고, scaled VLA representation이 이를 robot execution에 맞게 흡수하는지 보는 실험이다.

## Experiments and Results

공식 article은 human demonstration에만 존재하는 scenario에서 robot policy를 평가했다. 예시는 다음과 같다.

| scenario | robot data | human data가 주는 정보 |
| --- | --- | --- |
| sorting eggs | robot data는 egg를 carton에 넣는 행동을 포함 | 색이 다른 egg를 여러 carton에 어떻게 분류할지 보여줌 |
| dresser tidying | diverse bedroom robot scenes | target dresser에서 jewelry, hair ties 등을 적절한 container에 배치하는 scene-specific rule |
| toolbox organization | robot manipulation data | tool/object category와 placement organization |
| sorting fruit | robot manipulation data | object category별 sorting rule |

핵심 결과는 human video를 넣은 co-finetuning이 4개 generalization scenarios에서 약 2x improvement를 보였다는 점이다. 공식 글은 이것을 emergent human-to-robot transfer로 해석한다. 특별한 transfer learning module 없이 pre-trained π0.5와 mixture co-training만으로 효과가 나타났기 때문이다.

## Scaling Analysis

이 연구의 더 중요한 부분은 scaling ablation이다. 논문/글은 “human data가 도움이 됐다”에서 멈추지 않고, robot pretraining diversity가 human data 흡수 능력을 어떻게 바꾸는지 본다.

| 관찰 | 의미 |
| --- | --- |
| pretraining scale이 커질수록 human data 추가 이득이 커짐 | human-to-robot transfer가 model/data scale에 따라 emergent하게 나타남 |
| sorting eggs에서 no-human fine-tuning은 약 60% pretraining dataset 이후 saturation | robot data만 늘려서는 해당 human-only scenario 정보를 더 못 얻는 구간이 있음 |
| human-video fine-tuning은 더 큰 robot pretraining에서 계속 향상 | robot data scale이 human data를 흡수하는 representation을 만든다는 해석 |
| T-SNE 분석 | final VLM backbone token을 mean-pool한 latent가 robot/human data 사이에서 점점 overlap됨 |

T-SNE 결과는 인과 증명이라기보다 representation diagnostic이다. 작은/no pretraining model은 human과 robot example이 latent에서 분리되고, 더 다양한 robot pretraining을 거친 model은 두 domain이 더 겹친다. 이는 performance 향상과 함께 관찰되는 evidence다.

## PI Family 안에서의 위치

이 연구는 PI family에서 data source 확장 축이다. `π0.5`가 web data, object detection, semantic subtask prediction을 co-training했다면, Learning from Human Videos는 human egocentric data도 scaled VLA가 활용할 수 있는지 확인한다.

| 연결 문서 | 연결점 |
| --- | --- |
| [π0.5](../pi05-open-world-generalization/) | base model과 heterogeneous co-training philosophy |
| [π0.7](../pi07-steerable-model/) | broad behavior steering을 위한 richer data/prompt source |
| [Robot Olympics / π0.6](../moravec-robot-olympics/) | 새 skill을 fine-tuning으로 빠르게 얻는 model scaling 흐름 |
| [RLT](../rlt-efficient-online-rl/) | human video와 autonomous experience는 서로 다른 data-scaling route |

## 한계와 주의

- human video가 모든 robot task에 바로 도움이 된다는 주장이 아니다. 평가 scenario는 human data에만 있는 task-specific setting이다.
- 3D hand position label을 어떻게 안정적으로 얻는지, annotation pipeline 품질이 중요하다.
- T-SNE overlap은 representation evidence지만 causal proof는 아니다.
- human data는 task strategy를 주고, low-level execution은 여전히 robot data와 base VLA capability에 의존한다.
