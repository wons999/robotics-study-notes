## 문제의식

LLM은 “영어를 프랑스어로 번역”과 “JSON format으로 출력” 같은 능력을 조합할 수 있다. VLA는 semantic concept을 이해하더라도 “새 도구를 쓰기”, “새 appliance 조작”, “새 robot embodiment에서 같은 task 수행”처럼 skill을 recombine하는 compositional generalization을 충분히 보여주지 못했다.

π0.7의 목표는 다음이다.

| 목표 | 설명 |
| --- | --- |
| out-of-box generalist | task별 specialist fine-tuning 없이 다양한 dexterous task 수행 |
| compositional generalization | training에 없는 appliance/task를 prompt와 coaching으로 해결 |
| cross-embodiment transfer | data가 없는 robot/task 조합에서도 수행 |
| steerability | 무엇을 할지뿐 아니라 어떻게 할지 prompt로 제어 |

## Method: Diverse Multimodal Prompting

π0.7은 단순 text instruction만 쓰지 않는다. 공식 글은 generalization의 key가 diverse data를 naive하게 섞는 것이 아니라, behavior를 disambiguate하는 diverse context를 prompt에 넣는 것이라고 설명한다.

| Prompt modality | 역할 |
| --- | --- |
| diverse language | task와 sub-step 설명 |
| metadata | speed, quality, desired strategy 같은 “how” 정보 |
| control modality | data source/robot/control style 차이를 명시 |
| visual subgoal image | sub-step end state의 precise spatial layout 제공 |
| world-model-generated subgoal | test-time visual generalization을 돕는 synthetic subgoal |

이 구조 덕분에 suboptimal autonomous data도 “lower quality/lower speed” metadata로 구분해 training에 포함할 수 있다. 즉 π0.7은 좋은 data만 골라 쓰는 것이 아니라 heterogeneous data의 behavior style을 prompt로 조건화한다.

## Experiments and Results

### Compositional Task Generalization: Air Fryer

π0.7은 air fryer에 sweet potato를 넣는 task를 zero-shot으로 시도한다. 단순 prompt만으로는 reasonable attempt를 하지만 완전히 끝내지는 못한다. 사람이 step-by-step language coaching을 제공하면 훨씬 잘 수행한다. 여러 번 coaching한 뒤에는 high-level policy를 fine-tune해 language subgoals를 autonomously 생성하고, world model이 만든 visual subgoal image와 함께 π0.7에 제공해 task를 수행한다.

중요한 분석:

- training set에는 해당 mobile robot의 air fryer loading demonstration이 없다.
- 가까운 episode로 air fryer basket을 닫는 두 episode와 DROID Franka data가 발견되었다.
- 실제 수행은 이 source episodes와 morphology/setting이 다르므로, 행동 조합 능력의 초기 신호로 해석한다.

### Cross-Embodiment Laundry Folding

π0.7은 bimanual UR5e system에서 laundry folding을 수행한다. 이 robot은 training set에서 underrepresented embodiment이며, 해당 robot/task 조합의 laundry folding data는 없다. Source robot과 UR5e는 size, positioning, morphology가 달라 strategy도 달라져야 한다.

공식 글은 π0.7의 success rate가 source robot에서 folding data를 수집했던 expert teleoperators가 UR5e에서 처음 시도한 “zero-shot” success rate와 match한다고 설명한다. 이 teleoperators는 평균 375시간의 teleoperation experience를 가졌다.

### RECAP Specialist와 비교

π0.7은 RECAP으로 학습한 π*0.6 specialist가 수행하던 laundry folding, espresso making, box folding task를 단일 model로 수행한다. 공식 글은 autonomous data와 RL-trained specialist experience를 strategy metadata로 distill해, 단일 π0.7 model이 task-specific RECAP specialist와 비슷하거나 더 높은 throughput을 얻는다고 설명한다.

### Out-of-Box Skill Breadth

공식 페이지는 vegetable peeling, glass door cleaning with Windex 등 다양한 platform/task examples를 보여준다. 이는 controlled benchmark 하나보다 “single unified model의 breadth”를 강조하는 evidence다.

## 한계와 주의

- 공식 글은 broad capability demo와 qualitative evidence가 많고, full paper/PDF 수치 표는 별도 확인해야 한다.
- air fryer는 완전 zero-shot success라기보다 coaching과 high-level policy fine-tuning이 결합된 흐름이다.
- “specialist 수준” 비교는 normalized throughput 기준이며 task별 raw success/throughput table을 추가 확인해야 한다.
