# Spatially Anchored Tactile Awareness

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Spatially Anchored Tactile Awareness for Robust Dexterous Manipulation |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2510.14647 |
| 태그 | tactile, dexterous manipulation, forward kinematics, contact geometry |

## 한 줄 요약

SaTA는 tactile feature를 hand kinematic frame에 anchoring하여 contact geometry를 더 명확히 표현하려는 dexterous manipulation 방법이다.

## 방법론

raw tactile signal은 풍부하지만 “손의 어디에서 어떤 접촉이 일어났는가”가 명확하지 않을 수 있다. SaTA는 forward kinematics를 이용해 tactile feature를 hand coordinate에 배치하여 policy가 접촉 위치와 object geometry를 더 잘 이해하게 한다.

## T-Rex와의 관계

T-Rex는 force history와 deformation map을 사용하고, SaTA는 tactile feature의 spatial grounding을 강조한다. T-Rex를 더 깊게 이해할 때 “tactile representation은 시간 정보와 공간 정보를 모두 가져야 한다”는 관점을 보강해준다.

## 한계

- sensor 위치와 kinematic calibration이 정확해야 한다.
- hand가 바뀌면 tactile frame 설계를 다시 봐야 한다.
