## 한 줄 요약

OpenVLA는 대규모 robot demonstration으로 학습된 open-source VLA로, 이후 VLA 연구의 실험 기반을 넓힌 모델이다.

## 데이터셋과 모델 구조

Open X-Embodiment 계열의 robot demonstration을 사용한다. image-language input을 받아 robot action을 예측하는 구조이며, open model이라는 점이 중요하다.

## 이 논문의 위치

OpenVLA는 open-source VLA baseline family에 속한다. 논문 Survey에서는 “공개 모델과 공개 robot dataset을 사용해 어디까지 재현 가능한가”를 확인하는 기준점으로 두면 좋다. tactile, force, high-frequency contact correction은 OpenVLA 자체의 중심 주제는 아니므로, contact-rich 조작을 볼 때는 별도 tactile-aware policy와 비교해야 한다.
