## One-line Summary

OpenVLA is a 7B-parameter open-source VLA trained on a large robot demonstration mixture and designed for efficient fine-tuning to new manipulation tasks.

## Problem Setting

- Robot embodiment: multiple embodiments from Open X-Embodiment.
- Observation: image and language instruction.
- Action: robot actions represented for VLA training.
- Data: about 970k real-world robot demonstrations.
- Task: generalist manipulation and fine-tuning to new tasks.

## Core Idea

OpenVLA addresses a practical bottleneck in VLA research: many strong VLAs were closed. It provides an open model, training recipe, and fine-tuning path for robot manipulation.

## Method Details

- Architecture: Llama 2 language backbone with DINOv2 and SigLIP visual features.
- Training recipe: large-scale robot demonstration training with efficient adaptation options.
- Data: Open X-Embodiment-style robot demonstrations.
- Inference / control loop: image-language conditioned robot action prediction.

## What I Should Remember

- OpenVLA is a key baseline for open VLA experimentation.
- It is mostly visual-language-action, not tactile-reactive.
- Tactile-VLA-style papers can be viewed as attempts to extend this VLA family into contact-rich sensing.

## Limitations

- Tactile and force signals are not central.
- Discrete or tokenized action formulations can struggle with high-frequency dexterous correction.
- Fine-tuning remains necessary for many target domains.
