# FAST

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | FAST: Efficient Action Tokenization for Vision-Language-Action Models |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2501.09747 |
| 태그 | VLA, action tokenization, DCT, high-frequency control |

## 한 줄 요약

FAST는 high-frequency continuous robot action sequence를 frequency-space token으로 압축해 autoregressive VLA가 더 효율적으로 action을 예측하게 만든다.

## 이 논문의 위치

FAST는 robot action을 continuous flow로 생성하는 대신, action tokenization을 개선해 autoregressive route를 유지하는 접근이다. VLA에서 action을 token으로 볼지 continuous trajectory로 볼지 비교할 때 중요한 기준점이다.
