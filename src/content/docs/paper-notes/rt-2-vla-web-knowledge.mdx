# RT-2

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control |
| 연도 | CoRL, 2023 |
| 링크 | https://arxiv.org/abs/2307.15818 |
| 태그 | VLA, web knowledge, action tokenization, semantic generalization |

## 한 줄 요약

RT-2는 vision-language model을 robot trajectory와 함께 finetuning하여, web-scale semantic knowledge를 robot action으로 옮기는 VLA framing을 보여준 논문이다.

## 방법론

robot action을 text token처럼 표현하고, VLM이 image와 instruction을 보고 action token을 출력하게 한다. 이 아이디어가 이후 OpenVLA, pi_0, tactile-aware VLA 흐름의 출발점 중 하나다.

## T-Rex와의 관계

RT-2는 semantic generalization을 열었지만, tactile/contact-rich control 문제는 직접 다루지 않는다. T-Rex는 VLA framing을 물리적 접촉과 dexterous manipulation으로 확장하는 흐름에 있다.
