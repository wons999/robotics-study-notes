# RT-2

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control |
| 저자 | Anthony Brohan, Noah Brown, Justice Carbajal, Yevgen Chebotar, Xi Chen, et al. |
| 연도 | CoRL, 2023 |
| 링크 | https://arxiv.org/abs/2307.15818 |
| 태그 | VLA, web knowledge, action tokenization, semantic generalization |

## 한 줄 요약

RT-2는 vision-language model을 robot trajectory와 함께 co-finetuning하여, web-scale semantic knowledge를 robot action generation에 연결한 대표적인 VLA 논문이다.

## 이 논문의 위치

RT-2는 web-scale VLM knowledge를 robot action으로 연결한 대표적인 VLA 논문이다. 이후 robot foundation model을 읽을 때 “vision-language reasoning을 action policy로 어떻게 바꾸는가”라는 기본 문제 설정을 제공한다.

| 관점 | RT-2가 만든 기준 | 읽을 때 확인할 질문 |
| --- | --- | --- |
| 핵심 질문 | web-scale VLM knowledge를 robot action으로 옮길 수 있는가? | web knowledge가 실제 조작 판단에 얼마나 transfer되는가 |
| action 표현 | action을 text token처럼 표현 | token action이 continuous precision을 충분히 담는가 |
| sensory input | vision, language 중심 | tactile/force가 없을 때 어떤 failure가 남는가 |
| 강점 | semantic generalization | physical contact grounding은 별도로 필요한가 |

## 문제 설정

기존 robot policy는 robot data만으로 학습되기 때문에 semantic generalization이 약하다. RT-2는 Internet-scale vision-language pretraining을 robot control에 직접 연결하려 한다.

## 모델 구조와 방법론

RT-2는 robot action을 language token과 같은 형식으로 바꾼다. 그러면 VLM은 자연어 답변을 생성하듯 robot action token을 생성할 수 있다.

| 구성 | 역할 |
| --- | --- |
| pretrained VLM | web-scale visual-language knowledge 제공 |
| robot trajectory finetuning | image/instruction에서 action token으로 mapping 학습 |
| action tokenization | robot action을 text-like token sequence로 표현 |
| co-finetuning | web VQA 같은 task와 robot action task를 함께 학습 |

## 실험에서 봐야 할 점

- robot training data에 없는 object나 instruction을 얼마나 잘 처리하는가?
- chain-of-thought style reasoning이 robot action에 도움이 되는가?
- action tokenization이 continuous precision에 어떤 제약을 만드는가?

## 기억할 점

- RT-2는 “VLA가 무엇인가”를 잡는 기준점이다.
- RT-2의 action token 방식과 pi_0 계열 continuous flow action 방식을 비교하면 action representation의 흐름이 보인다.
- semantic reasoning이 좋아도 접촉 dynamics는 별도의 grounding이 필요하다.

## 한계

- tactile/force feedback은 중심이 아니다.
- tokenized action은 high-DoF continuous dexterous control에서 resolution 문제가 생길 수 있다.
- semantic reasoning이 좋아도 접촉 dynamics는 별도의 grounding이 필요하다.
