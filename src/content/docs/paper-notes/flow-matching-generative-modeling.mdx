# Flow Matching

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | Flow Matching for Generative Modeling |
| 연도 | ICLR, 2023 |
| 링크 | https://arxiv.org/abs/2210.02747 |
| 태그 | flow matching, generative modeling, continuous normalizing flow |

## 한 줄 요약

Flow Matching은 noise distribution에서 data distribution으로 이동하는 vector field를 직접 regression하여 sample을 생성하는 generative modeling 방법이다.

## 방법론

학습 시 simulation을 반복하기보다, path 중간 지점의 target velocity를 맞추는 방식으로 vector field를 학습한다. robotics에서는 action trajectory를 생성해야 할 sample로 보고 이 방법을 action generation에 사용할 수 있다.

## T-Rex와의 관계

pi_0와 T-Rex 모두 continuous action generation에 flow matching 계열 사고를 사용한다. T-Rex는 이 denoising path를 action expert와 tactile expert가 나누어 처리한다는 점이 중요하다.
