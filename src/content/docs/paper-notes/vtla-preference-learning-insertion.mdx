# VTLA

## 메타데이터

| 항목 | 내용 |
| --- | --- |
| 논문 | VTLA: Vision-Tactile-Language-Action Model with Preference Learning for Insertion Manipulation |
| 연도 | arXiv, 2025 |
| 링크 | https://arxiv.org/abs/2505.09577 |
| 태그 | tactile, VLA, insertion, preference learning |

## 한 줄 요약

VTLA는 insertion task에서 vision, touch, language, action을 함께 다루고 preference learning으로 action quality를 더 잘 학습하려는 모델이다.

## 데이터셋과 문제 설정

주요 task는 peg-in-hole이나 fingertip insertion처럼 접촉 정렬이 중요한 조작이다. simulation 기반의 vision-tactile-action-instruction 데이터를 사용해 real-world transfer를 노린다.

## 방법론

VLA 학습에서 단순 token prediction loss만으로는 continuous manipulation quality를 잘 표현하기 어렵다. VTLA는 preference learning을 사용해 더 나은 trajectory/action을 선호하도록 학습한다.

## 이 논문의 위치

VTLA는 insertion 계열 contact-rich task에서 tactile grounding과 preference learning이 어떤 역할을 하는지 보여준다. 정밀 삽입처럼 성공/실패 경계가 좁은 과제에서는 tactile signal과 평가 기준을 함께 설계해야 한다.
